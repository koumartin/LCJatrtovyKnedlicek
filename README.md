# LcJatrovyKnedlicek
![Thunderstore Version](https://img.shields.io/thunderstore/v/koumartin/LcJatrovyKnedlicek)

This mod adds a single scrap - "Játrový knedlíček". It is based on a passage from legendary czech reality show
"Ano šéfe!". You can watch the clip here: https://www.youtube.com/watch?v=Quf8F2n2jmc  

This mod is not meant to be taken seriously.
